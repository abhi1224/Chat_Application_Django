About Chat application

"Web based chatting application" is a web application. This web application is a way to communicate with new people which helps to exchange ideas, learn new things from others. This is a social-networking tool that leverages on technology advancement thereby allowing its users communicate and share ideas. It offers a wonderful one stop shop experience for keeping in touch with people you know. It can be used for messaging. It is named as a web based chatting application where a new user has to register an account in order to use the chat system. Some particular that user have to fill in such as the Username, password, first name, last name, and email. The username register for the system must be unique. After register it will direct to the login page. The user has to fill in the username and the password in order to login in the system. If the username or password incorrect, the system will inform the user the incorrect input and request user to type in again from where the user can select a user to start conversation.

HOW IT WORK

1.Register your account
2.Login with username and password
3.Select a user to start chat
4.Start messaging
